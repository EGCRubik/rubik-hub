Info: Fakenodo is a simulated repository service for development and testing purposes. It mimics the basic functionality of Zenodo without relying on the actual API.

API Endpoints

Method Endpoint Description
POST /fakenodo/deposit/depositions Create new record
GET /fakenodo/deposit/depositions List all depositions
GET /fakenodo/deposit/depositions/{id} Get deposition details
PATCH /fakenodo/deposit/depositions/{id}/metadata Update metadata (no new DOI)
POST /fakenodo/deposit/depositions/{id}/files Upload file
POST /fakenodo/deposit/depositions/{id}/actions/publish Publish (generates new DOI)
GET /fakenodo/deposit/depositions/{id}/versions List versions
DELETE /fakenodo/deposit/depositions/{id} Delete deposition
